UrbanStore

UrbanStore es una aplicación web desarrollada con **React + TypeScript** que simula una tienda de moda urbana.

Este repositorio corresponde a la **primera versión del proyecto**, centrada en el layout inicial y la página Home.

**En desarrollo**

Actualmente implementado:
- Header con logo
- Navbar con navegación básica
- Home con carrusel (autoplay + controles)
- Footer fijo al final de la página
- Estructura base del proyecto

-----------------------------------------------

Tecnologías utilizadas

- React
- TypeScript
- Vite
- CSS (layout y estilos personalizados)
- React Router DOM
- Git & GitHub
